How to Run (Examples)

You can view the help menu and all available flags by running:
`python cli.py --help`

Example 1: Placing a Market Order
`python cli.py --symbol BTCUSDT --side BUY --type MARKET --qty 0.01`

Example 2: Placing a Limit Order
`python cli.py --symbol BTCUSDT --side SELL --type LIMIT --qty 0.01 --price 80000`
